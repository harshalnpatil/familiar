# nest-client

Developer area for Familiar Nest client code (request routing, policy checks, and adapter execution).

This folder is intentionally scaffolded as the first step before implementing Nest client runtime modules.

Harness-specific local execution adapters live in `src/harness-adapters/`:

- `src/harness-adapters/adapters/codex.js`
- `src/harness-adapters/adapters/claude-code.js`
- `src/harness-adapters/runner.js`
